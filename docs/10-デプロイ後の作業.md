# 10 - デプロイ後の作業

## 全体の流れ

```
① Vercel にデプロイ
      ↓
② サイトURL・OGP画像URLを各ファイルに書き換え
      ↓
③ OGP画像を作成してアップロード
      ↓
④ X Card Validator で表示確認
      ↓
⑤ 広告設置 Phase 1（アフィリエイトリンク）
      ↓
⑥ 広告設置 Phase 2（Google AdSense 申請・設置）
```

---

## ① Vercel にデプロイ

### 1-1. GitHub リポジトリの確認

現在のリポジトリ (`ガチャカウンター/`) に `browser/` フォルダが含まれていることを確認する。

```
ガチャカウンター/
├── browser/
│   ├── index.html
│   ├── privacy.html
│   ├── css/style.css
│   └── js/app.js
├── obs/
└── docs/
```

### 1-2. Vercel にサインアップ・ログイン

1. [https://vercel.com/](https://vercel.com/) にアクセス
2. 「Continue with GitHub」で GitHub アカウントでログイン

### 1-3. 新しいプロジェクトを作成

1. ダッシュボードの「Add New」→「Project」をクリック
2. 対象の GitHub リポジトリを選択して「Import」
3. 設定画面で以下を入力：

| 項目 | 設定値 |
|---|---|
| Framework Preset | **Other**（静的HTMLのため） |
| Root Directory | **`browser`** |
| Build Command | （空欄のまま） |
| Output Directory | （空欄のまま） |

4. 「Deploy」をクリック
5. デプロイ完了後、`https://プロジェクト名.vercel.app/` のURLが発行される

## ② サイトURL・OGP画像URLを書き換え

デプロイ後に発行されたURLを以下の2ファイルに反映する。

### browser/index.html の OGP タグ

```html
<!-- 変更前 -->
<meta property="og:url"         content="https://example.com">
<meta property="og:image"       content="https://example.com/ogp.png">
<meta name="twitter:url"         content="https://example.com">
<meta name="twitter:image"       content="https://example.com/ogp.png">

<!-- 変更後（例） -->
<meta property="og:url"         content="https://gacha-counter.vercel.app">
<meta property="og:image"       content="https://gacha-counter.vercel.app/ogp.png">
<meta name="twitter:url"         content="https://gacha-counter.vercel.app">
<meta name="twitter:image"       content="https://gacha-counter.vercel.app/ogp.png">
```

### browser/js/app.js のツイートテキスト

```javascript
// 変更前
text += '\nガチャカウンターで記録中 → https://example.com';

// 変更後（例）
text += '\nガチャカウンターで記録中 → https://gacha-counter.vercel.app';
```

変更後は git にコミット・プッシュすると Vercel が自動で再デプロイする。

## ③ OGP画像を作成してアップロード

### 3-1. 画像の仕様

| 項目 | 推奨値 |
|---|---|
| サイズ | **1200 × 630 px** |
| ファイル形式 | PNG または JPG |
| ファイル名 | `ogp.png`（`browser/` 直下に置く） |
| ファイルサイズ | 5MB 以下 |

### 3-2. 作成ツール

- **Canva**（無料・日本語対応）: [https://www.canva.com/](https://www.canva.com/)
  - テンプレート検索: 「OGP」または「Twitter カード」
  - サイズ指定: 1200 × 630 px でカスタムサイズを作成
- **Adobe Express**（無料枠あり）でも作成可能

### 3-3. 画像の内容（推奨）

- ツール名「ガチャカウンター」を大きく表示
- キャッチコピー（例:「ガチャの回数・獲得履歴を記録できるカウンターツール」）
- ダークテーマのデザイン（サイトのデザインに合わせる）

### 3-4. アップロード方法

作成した `ogp.png` を `browser/` フォルダに置いて git にコミット・プッシュする。

```
browser/
├── index.html
├── privacy.html
├── ogp.png        ← 追加
├── css/style.css
└── js/app.js
```

## ④ X Card Validator で表示確認

OGP が正しく機能しているか X 公式ツールで確認する。

1. [https://cards-dev.twitter.com/validator](https://cards-dev.twitter.com/validator) にアクセス
2. サイトの URL（例: `https://gacha-counter.vercel.app`）を入力して「Preview card」
3. 画像・タイトル・説明文が正しく表示されているか確認する

**よくある問題:**

| 症状 | 原因・対処 |
|---|---|
| カードが表示されない | デプロイ直後はキャッシュに時間がかかる。数分待って再試行 |
| 画像が表示されない | `og:image` のURLが正しいか確認。画像ファイルが公開されているか確認 |
| 古い内容が表示される | Validator の「Invalidate cache」ボタンでキャッシュをクリア |

## ⑤ 広告設置 Phase 1（アフィリエイトリンク）

`file://` でもブラウザでも動作する。デプロイ前から設置可能。

### A8.net の場合

1. [https://www.a8.net/](https://www.a8.net/) に登録・ログイン
2. 「広告プログラム」からゲーム課金・スマホゲーム系の案件を検索
3. 提携申請を送り、承認後「広告タグ取得」からバナーの HTML タグをコピー
4. `browser/index.html` の `<div class="ad-area" id="adArea">` 内に貼り付ける

### Amazonアソシエイトの場合

1. [https://affiliate.amazon.co.jp/](https://affiliate.amazon.co.jp/) に登録・審査通過
2. 「リンクを作成」からゲーム関連商品のバナー HTML を取得
3. 同じく `<div class="ad-area" id="adArea">` 内に貼り付ける

## ⑥ 広告設置 Phase 2（Google AdSense）

### 6-1. AdSense 申請前に準備するもの

AdSense の審査を通過するために以下を確認・対応する。

| 項目 | 状況 | 対応 |
|---|---|---|
| プライバシーポリシー | ✅ `privacy.html` 作成済み | フッターリンクから確認できることを確認 |
| サイトが HTTPS で公開 | ✅ Vercel は自動で HTTPS | 不要 |
| オリジナルコンテンツ | ツールのみだと薄いと判断される場合あり | 使い方説明・更新履歴などのコンテンツ追加を検討 |
| お問い合わせ手段 | 未設置 | メールアドレスや X アカウントのリンクを `privacy.html` に追記することを推奨 |
| PV数 | 明確な基準なし | 一定のアクセスがあると承認されやすい |

### 6-2. AdSense に申請

1. [https://www.google.com/adsense/](https://www.google.com/adsense/) にアクセス
2. Google アカウントでログイン
3. サイト URL（例: `https://gacha-counter.vercel.app`）を入力して申請
4. 審査コードを `browser/index.html` の `<head>` 内に貼り付けてデプロイ（審査用）
5. 審査結果はメールで通知（数日〜数週間かかる場合あり）

### 6-3. 審査通過後：AdSense コードを設置

**自動広告（`<head>` に1行追加）:**

```html
<head>
  ...
  <!-- Google AdSense -->
  <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js?client=ca-pub-XXXXXXXXXX"
     crossorigin="anonymous"></script>
</head>
```

**ディスプレイ広告（広告エリアに設置）:**

```html
<div class="ad-area" id="adArea">
  <ins class="adsbygoogle"
       style="display:block"
       data-ad-client="ca-pub-XXXXXXXXXX"
       data-ad-slot="XXXXXXXXXX"
       data-ad-format="auto"
       data-full-width-responsive="true"></ins>
  <script>
       (adsbygoogle = window.adsbygoogle || []).push({});
  </script>
</div>
```

`ca-pub-XXXXXXXXXX` と `data-ad-slot` の値は AdSense 管理画面から取得する。
