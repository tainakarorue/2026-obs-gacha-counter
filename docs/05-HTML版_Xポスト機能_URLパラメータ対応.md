# 05 - HTML版 Xポスト機能 URLパラメータ対応

## 概要

ゲーム名・ハッシュタグはコードに直接書かず、URLパラメータで外部から渡す。
これにより、同じ `index.html` を複数ゲームで使い回せる。

---

## 対応するURLパラメータ

| パラメータ名 | 型 | 説明 | 省略時の動作 |
|---|---|---|---|
| `gameName` | 文字列 | ゲーム名。ツイートのヘッダーとハッシュタグに使用 | `【ガチャ結果】` のみ表示、ゲーム名ハッシュタグなし |
| `hashtags` | 文字列（カンマ区切り） | 追加ハッシュタグ（`#` は不要） | `#ガチャカウンター` のみ付与 |

---

## 設定方法

### 通常のブラウザで開く場合

アドレスバーに以下のように入力する。

```
file:///C:/Users/miz32/Desktop/ガチャカウンター/index.html?gameName=ブルアカ&hashtags=ブルーアーカイブ,ガチャ
```

### OBSブラウザソースで設定する場合

OBSの「プロパティ」→「URL」欄に入力する。
（OBS上ではXポストは動作しないが、パラメータは設定しておくと配信後にブラウザで開いた際に有効になる）

```
file:///C:/Users/miz32/Desktop/ガチャカウンター/index.html?gameName=ブルアカ&hashtags=ブルーアーカイブ,ガチャ
```

---

## 設定例と生成されるツイート

### 例1: gameName と hashtags を両方設定

```
?gameName=ブルアカ&hashtags=ブルーアーカイブ,ガチャ
```

生成されるツイート:

```
【ブルアカ ガチャ結果】
総回数: 183回 / 天井: 200回

ホシノ ×2 1凸（合計94回）
アビドス ×1 0凸（合計47回）

#ガチャカウンター #ブルアカ #ブルーアーカイブ #ガチャ
```

---

### 例2: gameName のみ設定

```
?gameName=原神
```

生成されるツイート:

```
【原神 ガチャ結果】
総回数: 90回 / 天井: 90回

胡桃 ×1 0凸（合計90回）

#ガチャカウンター #原神
```

---

### 例3: パラメータなし

```
index.html（パラメータなし）
```

生成されるツイート:

```
【ガチャ結果】
総回数: 47回 / 天井: 200回

キャラクター ×1 0凸（合計47回）

#ガチャカウンター
```

## JS実装コード（03の再掲）

`getUrlParams()` がボタンクリック時に毎回URLを読み取るため、
ページをリロードせずにURLを変更した場合でも反映される。

```javascript
function getUrlParams() {
  const params = new URLSearchParams(window.location.search);
  return {
    gameName: params.get('gameName') || '',
    hashtags: params.get('hashtags') || '',
  };
}
```

## OBSでの運用フロー

```
配信中 (OBSブラウザソース)
    → ガチャを引いて記録する
    → 「X にポスト」ボタンは表示されているが押しても動作しない

配信終了後 (通常ブラウザ)
    → 同じURLをChromeまたはEdgeで開く
    → localStorage にデータが保存されているため記録が残っている ※注意点参照
    → 「X にポスト」ボタンを押すと投稿画面が開く
    → 内容を確認してポスト
```

### ※ OBSとブラウザでlocalStorageが共有されない問題

OBSブラウザソースのlocalStorageと通常ブラウザのlocalStorageは**別々に管理**される。
そのため、配信中の記録をブラウザ側で確認するには以下のいずれかの対応が必要。

**回避策A（推奨）: 配信前にブラウザで開いて記録を始め、OBSは表示専用にする**

```
Chrome/Edge で index.html を開いて操作 → 記録はブラウザのlocalStorageに保存
OBSは同じファイルをブラウザソースで表示するだけ（操作はしない）
配信終了後、そのままブラウザでポストボタンを押す
```

**回避策B: 配信終了後にOBSのブラウザソースをリフレッシュしてポスト**

```
OBS上で「X にポスト」ボタンを押すと、OBSの内蔵ブラウザが投稿画面を開こうとする
（OBSのバージョンや設定によっては動作する場合もある）
```
